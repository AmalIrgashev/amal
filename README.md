# AMAL TV - Professional Platforma

Premium kontentlar va jonli TV kanallarni tomosha qilish uchun professional veb-platforma.

## Xususiyatlari
- **Jonli TV Kanallar**: O'zbekiston 24, Zo'r TV, Sevimli TV (HLS/M3U8 qo'llab-quvvatlaydi).
- **Premium Tizim**: Obuna bo'lish orqali premium kontentlarni ochish imkoniyati.
- **Login/Parol**: Foydalanuvchi ma'lumotlarini xavfsiz saqlash.
- **Sevimli Videolar**: Sevimli kontentlarni alohida ro'yxatga qo'shish.
- **Modern Dizayn**: Outfit shrifti va qorong'u rejim (Dark Mode).

## Texnologiyalar
- HTML5 / CSS3 / JavaScript
- [Hls.js](https://github.com/video-dev/hls.js/) - IPTV oqimlar uchun.
- Font Awesome - Ikonkalar uchun.
- Google Fonts (Outfit) - Tipografiya uchun.

## O'rnatish va Deployment
1. Shunchaki `index.html` faylini brauzerda oching.
2. **GitHub Pages**: Settings -> Pages -> main branch.
3. **Vercel**: GitHub repozitoriyasini Vercel-ga ulang va bir zumda jonli havola oling.

## Vercel Deployment Qadamlari
1. [Vercel.com](https://vercel.com) saytiga kiring.
2. **"Add New Project"** tugmasini bosing.
3. GitHub orqali `AmalIrgashev/AmalIrgashev` repozitoriyasini tanlang va **"Import"** bosing.
4. Hech qanday sozlamani o'zgartirmasdan **"Deploy"** tugmasini bosing.
